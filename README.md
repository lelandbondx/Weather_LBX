# Pixel Weather Bot

A retro GameBoy Color-style weather viewer with fun pixel animations and CRT vibes. Built with HTML, CSS, and JS. Fetches live weather from OpenWeatherMap for cities like Boston.

## How to Use
- Open the live site: https://lelandbondx.github.io/Weather_LBX/
- Enter a city (e.g., Boston) and click "GO!" to see the weather with animations like pulsing suns, falling rain with splashes, twirly snow, flashing lightning, or drifting fog.
